# Business Analytics Dashboard

This project presents a dashboard created using Tableau to analyze business data.

## 🔗 Tableau Dashboard Link

[Click here to view dashboard](https://public.tableau.com/app/profile/ramya.gupta1364/viz/businessanalyticsdashboard_17771127967290/Businessanalyticsdashboard?publish=yes)

## 📊 Features

* Sales and Profit Analysis
* Category-wise distribution (Pie Chart)
* Region-wise performance (Bar Chart)
* Sales Trend (Line Chart)

## 🛠 Tools Used

* Tableau Public
* Microsoft Excel
